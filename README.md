# 📝 BlogPost Website

A modern blog posting web application where users can create, read, update, and delete blog posts. Built with a focus on simplicity, performance, and clean UI.

---

## 🚀 Features

- ✍️ Create and publish blog posts
- 📖 Read all published posts
- 🛠️ Edit and update existing posts
- 🗑️ Delete posts
- 🔍 View single blog post page
- 👤 User authentication 


---

## 🧰 Tech Stack

- **Frontend:** React
- **State Management:** Redux 

- **Database:** Appwrite 
- **Styling:** Tailwind CSS
